# Basic Book CRUD Operation

----------------------------------------------------------------------
A basic CRUD Operation to practice Golang

### Tools

---------------------------------------------------------------------
- Golang
- Echo
- Gorm
- SQLite

### API

----------------------------------------------------------------------------

```
GET /books
POST /books
GET /books/{id}
PUT /books/{id}
DELETE /books/{id}
```
